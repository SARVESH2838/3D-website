# Techfest CA Nexus - Multi-page HTML Structure

This is a creative multi-page structure for a Techfest IIT Bombay College Ambassador style 3D website.

## Pages
- `index.html` - Landing page
- `tasks.html` - Task / mission page
- `dashboard.html` - Ambassador dashboard preview
- `rewards.html` - Rewards and leagues
- `projects.html` - Campaign/project tracking
- `store.html` - Credits/store module
- `contact.html` - FAQ and contact

## Assets
- `css/styles.css` - Shared styling
- `js/main.js` - Shared navbar, FAQ, parallax and background animation
